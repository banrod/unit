# Benchmark Results Archive

Use this folder to store run artifacts from `ImageFilterStore` benchmarks.

- Create a subfolder per run date/time (for example, `2025-02-19T1030Z/`).
- Include the input parameters (tag density, root distribution, hardware info).
- Save raw JSON metric dumps plus a short summary Markdown describing key takeaways.

Keeping results side-by-side with the plan in `../benchmark-plan.md` preserves context for future coherence reviews and regression tracking.
