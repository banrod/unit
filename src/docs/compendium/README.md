# Compendium: Frugality of the Greater Bodies

This compendium explores how large systems and natural formations achieve efficiency through restraint. It is organised into numbered chapters with supporting notes and references.

* **00_intro/** – background material and goals.
* **01_sections/** – main chapters of the compendium.
* **02_references/** – bibliography and source material.
* **assets/** – shared figures or diagrams used across chapters.

A build script under `scripts/` can merge the chapters into a single Markdown file for distribution.
